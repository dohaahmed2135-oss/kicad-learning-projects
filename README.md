# KiCad Learning Projects

This repository documents my journey of learning PCB design using KiCad.

The goal of these projects is to develop practical skills in:
- Schematic capture
- PCB layout and routing
- Component footprint selection
- Design rule checking (DRC/ERC)
- Hardware design for embedded applications
